3. Windows Firewall Freigabe manuell hinzufügen

Wenn das Popup zur Freigabe der Zulu Platform Architecture in der Windows Firewall nicht erscheint, kann die Freigabe manuell hinzugefügt werden.

Öffne ein Windows Terminal mit Administratorrechten und führe den folgenden Befehl aus: (rechtsklick auf das Windows-Logo in der Taskleiste, wähle Terminal (Admin) oder Windows PowerShell (Admin)

Füge den nachfolgenden Befehl ein und drücke Enter.
Wenn der Befehl erfolgreich ausgeführt wurde, sollte einfach nur OK erscheinen.

netsh advfirewall firewall add rule name="appleJuice" dir=in action=allow protocol=tcp localport=9850
Stelle sicher, dass der Port 9850 Am Ende mit deinem Core-Port übereinstimmt.
