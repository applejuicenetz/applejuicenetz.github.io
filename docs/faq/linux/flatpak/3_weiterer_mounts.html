2.3. weitere Ordner im Core Flatpak erlauben

Standardmäßig hat das appleJuice Core Flatpak nur Zugriff auf:

  • ~/appleJuice/
  • /media
  • /mnt
  • /run/media

Weitere Ordner können mit dem folgenden Befehl hinzugefügt werden:

flatpak override --filesystem=/path/to/folder io.github.applejuicenetz.core
