Die Integration für Home Assistant ermöglicht es, viele Informationen des appleJuice Core als Sensoren zu nutzen.

So können z.B. deine Credits, die Anzahl der Uploads, Downloads, die aktiven Verbindungen und vieles mehr in Home Assistant angezeigt und in Automatisierungen verwendet werden.

Die Integration verhält sich wie all die anderen appleJuice GUI. Daher, gib bei der einrichtung die gleichen Daten ein, wie du es in einem appleJuice GUI tun würdest.
